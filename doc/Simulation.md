# Simulation
TMS simulation and visualization module. The Simulation module provides realtime TMS visualization and control functionality specifically tailored for robotic TMS applications.

## Features
- TMS coil and electric field visualization in the real-time
- Simulation of electric field generated by TMS stimulation
- Data logging and export for analysis
- Free coil placement and controlled coil placement

## How to
1. Load the mrb data file that contains the skin, brain and target points
2. Go to Simulation module, in Data Directory section, browse to pick the folder that contains the conductivity file, the magfield file, and the coil cad model file. 
3. Run the server in a separate terminal to host the deep learning model. The code is in SlicerTMS-origin/server/server.py, you can run it with the command below:
```bash
python3 server.py
```
4. In Simulation module, click "Load Example" button to pass the data path to the server, and the server will run as soon as it receives a new incoming magfield data.
